================ INSTALLATION ================

To install Easy Content Types, simply upload the unzipped folder called "easy-content-types" to your wp-content/plugins directory.

Next, navigate to your WordPress plugins menu and click "Activate" on "Easy Content Types".

You will now have a new menu item called "Content Types". This is where you will create new post types, taxonomies, and meta boxes.

================ USAGE =======================

For your convenience, all usage documentation has been included inside WordPress.

Click on the "Help" link inside the "Content Types" menu.
